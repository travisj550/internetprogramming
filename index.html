Assignment 3

Unit Testing Results

This assignment is a calculator that finds the count, sum, mean, median, mode, variance and standard deviation for a list of numbers. It creates an array from that list and then calls other functions to calculate statistics.

By passing those statistical values back to subsequent functions, I was able to simplify my code. Standard deviation is the square root of variance, for instance, and both involve the count and sum.

  • Learn to use the output of functions in other functions
  • Understand how to do basic unit testing
  • Get a basic idea of how JavaScript libraries can be used

Assignment 4

Assignment 4 is an animation of US employment by sector (Industry, Services, and Agriculture) from 1991 to 2015. A different chart cycles in every 2 seconds to illustrate change over time.

For the chart animations, I used the Google Charts library. This simplified some things, but also required different syntax and data structures than I am used to. Additionally, I ran into an issue where the Google Charts library wasn't loading early enough for my chart to display unless the page was refreshed. The google.charts.setOnLoadCallback(function) function resolved this issue by preventing the functions requiring Google Charts from being called until after the Charts API is loaded.

Objectives:

  • Learn to use animation to illustrate an idea
  • Use libraries to simplify code
  • Get a basic idea of how animations can be coded

Assignment 6

Entity Relationship Diagram

Assignment 6 is a modifiable table of CPUs. Information is pulled from a mySQL database through PDO. User inputs to the database are only executed from prepared statements.

One thing that stood out to me, again: moving between HTML, PHP and Javascript adds complexity to webpages, but ultimately is a more powerful way to program. I wasn't able to access PHP error logging in this assignment, adding to that complexity. I think PHP error logging may have the potential to greatly accelerate future work in PHP.

Assignment 7

Assignment 7 adds a text search box to Assignment 6. As text is entered, the CPU table is filtered based on terms or values.

Underlying that is an html form that runs a Javascript function as input is typed with the onkeyup attribute. That function sends input to a PHP script with an XMLHttpRequest object. In PHP I used prepared statements with PDO, again. My SQL queries use the LOCATE function to find rows containing user input as substrings. Once the Javascript function gets a response from the server, the response text is then used to hide or reveal rows in the CPU table.

Objectives:

  • Filter assignment 6 table using AJAX and PHP
  • Sanitize user inputs
  • Keep formatting consistent
